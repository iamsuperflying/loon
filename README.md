# Loon Plugins

个人 Loon 插件仓库。

## 目录结构

```
.
├── plugins/    # 插件文件 (.plugin)
├── scripts/    # JavaScript 脚本
├── rules/      # 分流规则
└── icons/      # 图标资源
```

## 使用方法

在 Loon 中添加插件，URL 格式：

```
https://raw.githubusercontent.com/iamsuperflying/loon/main/plugins/<插件名>.plugin
```
